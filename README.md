## 🚀 Visão Geral

O **CertGuard AI** é um sistema completo e avançado de gerenciamento de certificados digitais que unifica as melhores funcionalidades de Presto/Oystr, Whom/Doc9 e LoyTrust, adicionando tecnologias de ponta como **IA NVIDIA**, **blockchain** e **análise comportamental avançada**.

### ✨ Principais Características

- 🤖 **LucIA - IA Jurídica Avançada** com NVIDIA API
- 🔐 **Gestão Completa de Certificados A1/A3**
- 🏛️ **Automação de Tribunais** (TJ-RJ, TJSP, TRF-2, PJe)
- 🔍 **Sistema de Auditoria e Análise Comportamental**
- 🌐 **Extensão Chrome/Edge** para download e automação
- 📊 **Dashboard Avançado** com métricas em tempo real
- 🔗 **Blockchain** para registros imutáveis
- 📱 **Interface Responsiva** e moderna

## 📦 Instalação

### Pré-requisitos
- Python 3.11+
- Node.js 18+
- Git

### 1. Clone o Repositório
```bash
git clone https://github.com/fernandolisboaneto/CERTGUARD-AI-100-.git
cd CERTGUARD-AI-100
```

### 2. Configurar Backend
```bash
cd backend
pip install -r requirements.txt

# Configurar variáveis de ambiente
export NVIDIA_API_KEY="sua_chave_nvidia"
export NVIDIA_API_BASE="https://integrate.api.nvidia.com/v1"

# Iniciar servidor
python src/main.py
```

### 3. Configurar Frontend
```bash
cd frontend
python -m http.server 8080
```

### 4. Instalar Extensão
1. Abra Chrome/Edge
2. Vá em **Extensões** → **Modo Desenvolvedor**
3. Clique em **Carregar sem compactação**
4. Selecione a pasta `extension/`

### 5. Docker (Opcional)
```bash
docker-compose up -d
```

## 🔧 Configuração

### Variáveis de Ambiente
```bash
# API NVIDIA
NVIDIA_API_KEY=nva...82_
NVIDIA_API_BASE=https://integrate.api.nvidia.com/v1
NVIDIA_MODEL=meta/llama-3.3-70b-instruct

# Banco de Dados
DATABASE_URL=sqlite:///certguard.db

# Segurança
JWT_SECRET_KEY=sua_chave_secreta
ENCRYPTION_KEY=sua_chave_criptografia

# Blockchain
BLOCKCHAIN_NETWORK=ethereum
BLOCKCHAIN_CONTRACT=0x...
```

### Configuração da LucIA
```python
LUCIA_CONFIG = {
    "model": "meta/llama-3.3-70b-instruct",
    "temperature": 0.2,
    "top_p": 0.7,
    "max_tokens": 1024,
    "stream": True
}
```

## 📖 Uso

### Dashboard Principal
1. Acesse a URL do sistema
2. Faça login com suas credenciais
3. Explore as funcionalidades no menu lateral

### Upload de Certificados (Admin)
1. Acesse **Certificados** → **Upload**
2. Selecione os arquivos .pfx/.p12
3. Configure permissões de acesso
4. Confirme o upload

### Download via Extensão
1. Instale a extensão CertGuard AI
2. Acesse um tribunal suportado
3. Clique no ícone da extensão
4. Selecione o certificado desejado
5. Faça o download

### Consultas à LucIA
1. Acesse **Segurança** → **LucIA**
2. Digite sua pergunta sobre auditoria
3. Receba análises inteligentes
4. Explore insights de segurança
